# Todd Smith - Google Search Simulation

A Google search simulation for **Todd Smith** showing low discoverability (cluttered results with many different people).

## Features

- **Google Search Results**: 5 pages of cluttered search results for "Todd Smith"
- **Low Discoverability**: Shows 2,700+ results with many different people named Todd Smith
- **Ambiguous Results**: Multiple profiles with same name, making it hard to identify the correct person
- **Various Locations**: Results from different cities and states across the US
- **Different Professions**: Software engineers, teachers, managers, accountants, etc.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to see the simulation.

## Todd Smith - Low Discoverability

- **Search Type**: Low Discoverability + Footprint Present
- **Result Count**: 2,700+ profiles
- **Challenge**: Many different people with the same name, making it difficult to identify the correct Todd Smith
- **Characteristics**: Cluttered results, ambiguous matches, multiple locations and professions
